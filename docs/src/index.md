# UnsteadyKineticRotorDynamics.jl

UnsteadyKineticRotorDynamics provides unsteady kinetic rotor dynamics for wind
and hydrokinetic rotors. It is built around low-pass time-domain dynamics on top
of steady CCBlade solves for studies where a full unsteady solver is not
required.
